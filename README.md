# python_security_log_analyzer


Overview

Security Log Analyzer is a Python-based cybersecurity project that analyzes login logs, detects failed login attempts, and identifies suspicious IP addresses.

Features

- Count successful login attempts
- Count failed login attempts
- Detect the most suspicious IP address
- Generate a security report file

Technologies Used

- Python
- File Handling
- Dictionaries
- Conditional Statements
- Loops

Example Logs Entry

192.168.1.1 LOGIN SUCCESS

10.0.0.5 LOGIN FAILED

Example Output

Successful logins: 2

Failed logins: 3

Most suspicious IP: 10.0.0.5

Failed attempts: 2

How to Run

1. Clone the repository
2. Open the project folder
3. Run:

python_security_log_analyzer

4. The program will generate a report.txt file containing the analysis results.
